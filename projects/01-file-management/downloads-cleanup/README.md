# Downloads Folder Cleanup

**Level**: Beginner
**Time**: 5-10 minutes
**Outcome**: Clean, organized Downloads folder with significant space savings

## What You'll Accomplish

Transform a chaotic Downloads folder with 1000+ files into an organized, clean directory structure. Free up gigabytes of wasted space from temp files and duplicates.

## The Professional Approach

### What You Say
```
"Organize my Downloads folder"
```

or

```
"Quick clean my Downloads - remove temp files and duplicates"
```

### What Claude Does

1. **Analyzes** the entire folder structure
2. **Identifies** problems (temp files, duplicates, scattered organization)
3. **Proposes** an organization strategy
4. **Executes** cleanup safely
5. **Reports** what was done and space saved

### Tools Claude Uses (Automatically)

- **Glob**: Find all files and folders
- **Bash**: Check file sizes, delete items, create folders
- **Read**: Analyze file lists
- **Write**: Create organization plans and reports
- **TodoWrite**: Track cleanup progress

**You don't specify these** - Claude chooses the right approach.
